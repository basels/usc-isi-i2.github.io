INTRODUCTION

Common sense reasoning (CSR) has been a moonshot problem in AI for over 50 years. Recently, with the advent of transformer-based neural networks pretrained on large quantities of text, with potentially billions, or even trillions of parameters, much progress has been made on building machines with common sense capabilities. These transformer-based 'language representation models' (LRMs) come in both generative (such as GPT-3) and discriminative (e.g., BERT) varieties. Modern LRMs released in the last 5 years have progressively achieved and pushed state-of-the-art performance on publicly available CSR benchmarks modeled as multiple-choice question answering tasks, with examples of such benchmarks including Physical IQA, abductive Natural Language Inference, Social IQA, HellaSwag, and many others.

[x] TG-CSR uses a theory of common sense developed in a series of papers (and collected in a book) by the researchers [x].

In particular, TG-CSR is designed to test the performance of CSR systems (whether LRM-based or not) in few-shot settings. Below, we provide an example training dataset with ground-truth labels using both the formal JSON format, as well as an excel format to facilitate easy browsing. We also provide an unlabeled test set that should be used for submissions. The format expected for submissions is detailed under Submissions, with an example submission file (containing random labels) provided as well. Additionally, we provide a link to a validation script for ensuring that the submission file is properly formatted. Upon submission, we will evaluate your entry using withheld ground-truth labels and place it in the correct spot on the leaderboard within 24 hours (of submission).

TG-CSR is a collaboration between researchers from the University of Southern California, Rensselaer Polytechnic Institute, and the University of California at Irvine.
